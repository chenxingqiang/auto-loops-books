Teams hit \textbf{sync hierarchy} when Hierarchical Synchronization and Communication looks correct in isolation yet decode telemetry disagrees---usually because sync was planned without barrier residency on the SKU you ship \citep{dao2022flashattention,yirage2026}.

Hardware truth: GPUs bound sync with shared memory/TMA; CPUs with cache/NUMA; NPUs with static tile SRAM and DMA chains \citep{nvidia2022hopper,amd2024xdna,dlbook}. Decode at batch-1 stays bandwidth- and orchestration-limited---thin matmuls rarely dominate \citep{memoryfloor2026,taxbreak2026}.

Compiler/kernel fix: predicate fusion on sync/barrier legality before codegen; fork passes per target \citep{chen2020compilerbasedhardw,yirage2026}. YiRage encodes iron rules from Chapter~2 as checkable IR metadata \citep{kwon2023vllm,liu2024deepseekv3}.

\paragraph{Multi-hardware delta.} Hopper may legalize sync with TMA double-buffer; XDNA requires static tile assignment; CPU needs cache-blocked barrier---identical math, different bytes/token \citep{semianalysis2024tma,amd2024xdna,dlbook}.

\end{example}

The production surprise at \textbf{hardware sync pick} is not peak FLOPs; it is illegal tier placement: GPU spilling to HBM/DDR while CPU still pays launch or DMA tax every token \citep{amd2024xdna,amd2024aie}.

Trace residency: GPU producers must feed CPU consumers without a DDR round trip \citep{dao2022flashattention,yirage2026}. Silent spill reads as ``mysterious'' p99 regression \citep{patel2025llminference,semianalysis2024tma}.

Operational fix: diff IR or CUDA for new allocs; count launches/token (GPU) or DMA edges (NPU); require graph capture legality on GPU and static legality on XDNA \citep{nvidia2024cudagraphs,amd2024aie}.

\paragraph{Review gate.} Attach roofline or buffer accounting for GPU on target SKU before merge \citep{memoryfloor2026,yirage2026}.
