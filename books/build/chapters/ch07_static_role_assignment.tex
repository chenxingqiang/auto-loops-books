The production surprise at \textbf{thread mapping} is not peak FLOPs; it is illegal tier placement: thread spilling to HBM/DDR while warp still pays launch or DMA tax every token \citep{nvidia2022hopper,semianalysis2024tma}.

Trace residency: thread producers must feed warp consumers without a DDR round trip \citep{dao2022flashattention,yirage2026}. Silent spill reads as ``mysterious'' p99 regression \citep{patel2025llminference,semianalysis2024tma}.

Operational fix: diff IR or CUDA for new allocs; count launches/token (GPU) or DMA edges (NPU); require graph capture legality on GPU and static legality on XDNA \citep{nvidia2024cudagraphs,amd2024aie}.

\paragraph{Review gate.} Attach roofline or buffer accounting for thread on target SKU before merge \citep{memoryfloor2026,yirage2026}.
