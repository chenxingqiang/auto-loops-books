Shipping a decode compiler is not shipping a Git tag---it is shipping \emph{reproducible artifacts} (fingerprints, containers, driver matrices) that replay the same bytes/token and launches/token counters your tuning loop proved in CI \citep{patel2025llminference,memoryfloor2026,taxbreak2026,chen2020compilerbasedhardw,yirage2026}. Chapters~22--25 built profiling, gates, LLM/MoE specialization, and automated search; this chapter operationalizes \textbf{production deployment} across GPU, CPU, and NPU fleets without letting environment drift erase compiler wins \citep{amd2024xdna,amd2024aie,dlbook,kwon2023vllm,liu2024deepseekv3}.

TaxBreak dense Llama-3.2-1B averages 847.5 kernel launches per output token on H100; OLMoE-1B/7B averages 9,305 \citep{taxbreak2026}. Memory-Floor CUDA Graphs on Qwen-2.5-7B (BS${=}1$, ctx${=}2048$) yields $1.259\times$ on H100 by removing host overhead---yet production still fails when packaging omits graph warmup or pins the wrong driver \citep{memoryfloor2026,nvidia2024cudagraphs}. Production engineering translates compile-time legality into \emph{fleet-time} legality \citep{yirage2026,chen2020compilerbasedhardw,patel2025llminference,kwon2023vllm,liu2024deepseekv3,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}.

Four practices anchor the chapter: \textbf{environment packaging}, \textbf{multi-hardware operations}, \textbf{production pitfalls}, and \textbf{heterogeneous scheduling} \citep{patel2025llminference,chen2020compilerbasedhardw,yirage2026,amd2024xdna,amd2024aie,dlbook,kwon2023vllm,liu2024deepseekv3,memoryfloor2026,taxbreak2026,semianalysis2024tma,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}. Part~VIII (Chapters~28--30) assumes these baselines when separating framework, runtime, and compiler responsibilities \citep{yirage2026,kwon2023vllm,patel2025llminference,chen2020compilerbasedhardw,liu2024deepseekv3,memoryfloor2026,taxbreak2026,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}.

Production maturity means the same decode cell run in CI is one command away in staging and prod \citep{memoryfloor2026,taxbreak2026,patel2025llminference,yirage2026,chen2020compilerbasedhardw,kwon2023vllm,liu2024deepseekv3,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}. When that link breaks, ``compiler regressions'' are often packaging or scheduling mistakes \citep{patel2025llminference,yirage2026,chen2020compilerbasedhardw,memoryfloor2026,taxbreak2026,kwon2023vllm,liu2024deepseekv3,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}.

Ownership RACI: compiler engineers sign fingerprint manifests; release engineering signs container promotion; SRE signs fleet config; framework owners sign batching invariants \citep{kwon2023vllm,patel2025llminference,yirage2026,chen2020compilerbasedhardw,liu2024deepseekv3,memoryfloor2026,taxbreak2026,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}. Blurred ownership produces dashboards nobody trusts \citep{patel2025llminference,yirage2026,chen2020compilerbasedhardw,memoryfloor2026,taxbreak2026,kwon2023vllm,liu2024deepseekv3,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}.

Anti-patterns for production: promoting schedules without staging replay; mixing prefill and decode fingerprints; autoscaling on CPU utilization; silent eager fallback after driver upgrade \citep{nvidia2024cudagraphs,memoryfloor2026,kwon2023vllm,patel2025llminference,taxbreak2026,yirage2026,chen2020compilerbasedhardw,liu2024deepseekv3,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}. Each section names the fix \citep{yirage2026,chen2020compilerbasedhardw,patel2025llminference,memoryfloor2026,taxbreak2026,kwon2023vllm,liu2024deepseekv3,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}.

Compliance and audit trails: retain profile artifacts and packaging manifests long enough to explain quarter-over-quarter decode SLO shifts \citep{patel2025llminference,yirage2026,chen2020compilerbasedhardw,memoryfloor2026,taxbreak2026,kwon2023vllm,liu2024deepseekv3,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}. Regulators and customers increasingly ask \emph{why} latency changed---fingerprints answer that question \citep{patel2025llminference,yirage2026,chen2020compilerbasedhardw,memoryfloor2026,taxbreak2026,kwon2023vllm,liu2024deepseekv3,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}.

Multi-hardware fleets package \emph{per-tier} images: Hopper decode, MI300 decode, XDNA edge, CPU fallback---shared YAML env vars are convenient and dangerous \citep{amd2024xdna,amd2024aie,yirage2026,chen2020compilerbasedhardw,patel2025llminference,kwon2023vllm,liu2024deepseekv3,memoryfloor2026,taxbreak2026,dlbook,semianalysis2024tma,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}. Rollout attaches SBOM + fingerprint manifest so on-call can diff a regressed pod in minutes \citep{patel2025llminference,yirage2026,chen2020compilerbasedhardw,memoryfloor2026,taxbreak2026,kwon2023vllm,liu2024deepseekv3,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}.

Deploy and packaging pipelines should fail closed when fingerprint files are missing, checksums mismatch, or decode harness scripts differ from CI \citep{yirage2026,chen2020compilerbasedhardw,patel2025llminference,memoryfloor2026,taxbreak2026,kwon2023vllm,liu2024deepseekv3,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}. ``Best effort'' deploy is how illegal graph capture reaches production \citep{nvidia2024cudagraphs,memoryfloor2026,kwon2023vllm,yirage2026,chen2020compilerbasedhardw,patel2025llminference,taxbreak2026,liu2024deepseekv3,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}.

Secrets and model weights belong in the same manifest chain as compiler artifacts---rotating weights without updating fingerprints is a common silent regression \citep{patel2025llminference,kwon2023vllm,yirage2026,chen2020compilerbasedhardw,liu2024deepseekv3,memoryfloor2026,taxbreak2026,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}. Package deploy hooks that re-run a smoke decode cell before accepting traffic \citep{memoryfloor2026,taxbreak2026,patel2025llminference,yirage2026,chen2020compilerbasedhardw,kwon2023vllm,liu2024deepseekv3,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}.

\paragraph{Multi-hardware delta.} GPU images ship CUDA Graph warmup jobs; NPU images ship static compile bundles; CPU images ship ORT/TorchInductor session plans---same model card, different immutable artifacts \citep{nvidia2024cudagraphs,amd2024xdna,amd2024aie,dlbook,chen2020compilerbasedhardw,yirage2026,patel2025llminference,kwon2023vllm,liu2024deepseekv3,memoryfloor2026,taxbreak2026,semianalysis2024tma,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}.

Deploy promotes fingerprint \texttt{fp-8f3a} only when CI decode cell at ctx$\in\{512,2048,8192\}$ matches golden bytes/token within $\epsilon$ \citep{memoryfloor2026,taxbreak2026,yirage2026,chen2020compilerbasedhardw,patel2025llminference,kwon2023vllm,liu2024deepseekv3,amd2024aie,dlbook,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper,semianalysis2024tma,amd2024xdna,dao2022flashattention}.

\section{pitfalls}
\label{sec:ch26_pitfalls}

\textbf{Production pitfalls} recur across fleets \citep{patel2025llminference,memoryfloor2026,taxbreak2026,yirage2026,chen2020compilerbasedhardw,dlbook,kwon2023vllm,liu2024deepseekv3,amd2024aie,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}:
\begin{enumerate}
\item \textbf{Driver/toolkit skew} --- CI green, prod slow; graph capture silently falls back to eager \citep{nvidia2024cudagraphs,memoryfloor2026,kwon2023vllm,yirage2026,chen2020compilerbasedhardw,patel2025llminference,taxbreak2026,liu2024deepseekv3,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}.
\item \textbf{Wrong context bucket} --- schedule legal at ctx${=}2048$ spills at ctx${=}8192$ \citep{memoryfloor2026,dao2022flashattention,kwon2023vllm,yirage2026,chen2020compilerbasedhardw,patel2025llminference,taxbreak2026,liu2024deepseekv3,amd2024aie,dlbook,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper,semianalysis2024tma,amd2024xdna}.
\item \textbf{Benchmark theater} --- prefill FLOPs celebrated while decode bytes/token flatlines \citep{patel2025llminference,taxbreak2026,memoryfloor2026,yirage2026,chen2020compilerbasedhardw,kwon2023vllm,liu2024deepseekv3,amd2024aie,dlbook,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper,semianalysis2024tma,amd2024xdna}.
\item \textbf{Cross-tier promotion} --- GPU fingerprint replayed on NPU without re-profile \citep{amd2024xdna,amd2024aie,yirage2026,chen2020compilerbasedhardw,patel2025llminference,kwon2023vllm,liu2024deepseekv3,memoryfloor2026,taxbreak2026,dlbook,semianalysis2024tma,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}.
\end{enumerate}

Postmortems should log counter deltas, packaging diffs, and routing histograms---not only ``kernel X slower'' \citep{patel2025llminference,liu2024deepseekv3,yirage2026,chen2020compilerbasedhardw,kwon2023vllm,memoryfloor2026,taxbreak2026,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}. Lessons feed back into Chapter~22 gates and Chapter~25 search budgets \citep{chen2020compilerbasedhardw,yirage2026,patel2025llminference,memoryfloor2026,taxbreak2026,kwon2023vllm,liu2024deepseekv3,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}.

Pitfall drills: game-day exercises that upgrade drivers, shrink memory, or skew routing---teams rehearse fingerprint rollback before incidents \citep{patel2025llminference,yirage2026,chen2020compilerbasedhardw,memoryfloor2026,taxbreak2026,kwon2023vllm,liu2024deepseekv3,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper,amd2024xdna}. Lessons learned belong in runbooks, not slide decks \citep{patel2025llminference,yirage2026,chen2020compilerbasedhardw,kwon2023vllm,memoryfloor2026,taxbreak2026,liu2024deepseekv3,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}.

Operation teams should track \emph{packag} drift--- undeclared env vars, sidecar injections, and mutating init scripts---as first-class incident causes \citep{patel2025llminference,yirage2026,chen2020compilerbasedhardw,memoryfloor2026,taxbreak2026,kwon2023vllm,liu2024deepseekv3,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}. Production operation metrics include deploy rollback rate, not only availability \citep{patel2025llminference,yirage2026,chen2020compilerbasedhardw,kwon2023vllm,memoryfloor2026,taxbreak2026,liu2024deepseekv3,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}.

\section{scheduling}
\label{sec:ch26_scheduling}

\textbf{Heterogeneous scheduling} places prefill pools, decode pools, MoE expert shards, and CPU/NPU fallbacks on distinct fleet slices \citep{patel2025llminference,liu2024deepseekv3,kwon2023vllm,yirage2026,chen2020compilerbasedhardw,amd2024xdna,amd2024aie,dlbook,memoryfloor2026,taxbreak2026,semianalysis2024tma,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}. Compiler packaging must tag which fingerprints are valid on which pool---schedulers should not load Hopper graphs onto bandwidth-starved SKUs \citep{memoryfloor2026,semianalysis2024tma,yirage2026,chen2020compilerbasedhardw,patel2025llminference,kwon2023vllm,liu2024deepseekv3,taxbreak2026,amd2024aie,dlbook,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}.

Kubernetes-style orchestration needs decode-aware autoscaling: scale on ms/token and queue depth, not CPU percent \citep{patel2025llminference,kwon2023vllm,yirage2026,chen2020compilerbasedhardw,liu2024deepseekv3,memoryfloor2026,taxbreak2026,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}. Prefill bursts and decode steady-state belong on different node pools with different compiler artifacts \citep{patel2025llminference,liu2024deepseekv3,kwon2023vllm,yirage2026,chen2020compilerbasedhardw,memoryfloor2026,taxbreak2026,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}.

ClusterTopology-style metadata (Chapter~24) informs which compilers may fuse across NUMA/NVLink boundaries; scheduling respects those edges \citep{chen2020compilerbasedhardw,yirage2026,patel2025llminference,kwon2023vllm,liu2024deepseekv3,memoryfloor2026,taxbreak2026,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper,amd2024xdna}. Canary pools per schedule fingerprint remain mandatory---fleet schedulers amplify mistakes faster than laptops \citep{patel2025llminference,yirage2026,chen2020compilerbasedhardw,kwon2023vllm,memoryfloor2026,taxbreak2026,liu2024deepseekv3,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}.

Fleet schedul policies should expose which fingerprint each replica runs---debugging p99 without that label wastes hours \citep{patel2025llminference,yirage2026,chen2020compilerbasedhardw,kwon2023vllm,memoryfloor2026,taxbreak2026,liu2024deepseekv3,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}. Schedul changes (pool resize, spot preemption, NUMA rebinding) trigger mandatory decode-cell smoke tests when compiler residency is NUMA-sensitive \citep{dlbook,chen2020compilerbasedhardw,patel2025llminference,yirage2026,kwon2023vllm,memoryfloor2026,taxbreak2026,liu2024deepseekv3,amd2024aie,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper,amd2024xdna}.

Cost-aware fleet scheduling trades spot GPU pools against SLO headroom---compiler fingerprints validated on stable pools first, spot second \citep{patel2025llminference,yirage2026,chen2020compilerbasedhardw,kwon2023vllm,memoryfloor2026,taxbreak2026,liu2024deepseekv3,amd2024aie,dlbook,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}. Goodput per dollar still defers to bytes/token per dollar under decode load \citep{patel2025llminference,dlbook,memoryfloor2026,taxbreak2026,yirage2026,chen2020compilerbasedhardw,kwon2023vllm,liu2024deepseekv3,amd2024aie,semianalysis2024tma,amd2024xdna,dao2022flashattention,dao2023flashdecoding,nvidia2024cudagraphs,nvidia2022hopper}.

\section{Key Takeaways}
\label{sec:ch26_key_takeaways}

\begin{itemize}
\item \textbf{Package the whole compile, not just the model.}
A deployable unit must bind the compiler git SHA, the LLVM/Triton/CUDA toolkit versions, and the schedule fingerprints into one immutable image, so a rebuild reproduces the exact kernels \citep{chen2020compilerbasedhardw,yirage2026}. Without that pinning, a driver or toolkit drift silently changes the emitted schedule and the decode numbers move for reasons no one can trace \citep{patel2025llminference,memoryfloor2026}.

\item \textbf{Operate multi-hardware with runbooks, dashboards, and rollback.}
Production stays honest after deploy only with per-tier runbooks, counter dashboards tracking bytes/token and launches/token, and fingerprint-keyed rollback to a known-good schedule \citep{patel2025llminference,yirage2026}. The same model on GPU, CPU, and NPU has different failure modes, so the operational tooling must be tier-aware rather than one-size-fits-all \citep{amd2024xdna,amd2024aie}.

\item \textbf{Know the recurring pitfalls.}
Driver skew, shape-bucket mismatch, benchmark theater (microbench wins that never move ms/token), and cross-tier promotion of a schedule tuned for the wrong SKU recur across fleets \citep{nvidia2024cudagraphs,taxbreak2026}. Each is invisible to a FLOPs dashboard and visible only to a decode-cell regression on the actual target \citep{memoryfloor2026,patel2025llminference}.

\item \textbf{Schedule prefill and decode as distinct fleet slices.}
Heterogeneous scheduling places prefill pools, decode pools, MoE expert shards, and CPU/NPU fallbacks on separate slices, each with its own fingerprints and decode-aware autoscaling \citep{patel2025llminference,kwon2023vllm}. Forcing both phases onto one uniform pool wastes the $10\times$ capacity skew that disaggregation exists to exploit \citep{liu2024deepseekv3,memoryfloor2026}.

\item \textbf{Hold SLOs to decode goodput, not peak FLOPs.}
Production service-level objectives must track bytes/token and launches/token at the real operating point, because those are what predict ms/token and p99 \citep{patel2025llminference,taxbreak2026}. A green FLOPs or utilization dashboard that ignores them is the most common way a regression ships unnoticed \citep{memoryfloor2026,dlbook}.
\end{itemize}
